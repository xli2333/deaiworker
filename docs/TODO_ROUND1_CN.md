## 执行规则

1. 只按本文件推进，完成一项勾一项。
2. 本轮 Todo 未完成前，不切换到口头规划模式。
3. 每次实现后必须做真实运行或构建验证，不能只看代码。
4. 如果本轮做完后仍存在明确提升空间，必须新开下一轮中文 Todo，再继续推进。
5. 所有新增代码与文档统一使用 UTF-8，避免中文乱码。
6. 前端代码必须保持 JSX/TSX 可构建，不留下语法半成品。
7. 核心 Prompt 与规则资产只放后端或运行时资产目录，不暴露到前端源码。
8. 本项目当前只服务单文档去AI，不扩成多文档写作平台。

## 范围边界

### 本轮要做

1. 独立项目目录与独立依赖。
2. 单文档上传。
3. Gemini API Key 输入与验证。
4. 去AI诊断、修订、复检。
5. 最终审计诊断、修订、复检。
6. 终修与结果查看。
7. 产物落盘与基础导出。

### 本轮不做

1. 公众号排版与发布。
2. 多文档合并。
3. 配图。
4. 管理后台。
5. 多用户并发与鉴权体系。
